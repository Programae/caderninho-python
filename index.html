Esta página foi feita para ser executada offline.

Modo offline

  1. Pré-requisito: Python 3.4.x instalado.
  2. Faça o download da caderninho.
  3. Descompacte o arquivo zip e execute(*) o arquivo caderninho.py
  4. Aguarde alguns instantes que o navegador (browser) irá carregar a página.

    5. (*) Windows: duplo clique no arquivo caderninho.
    Linux: execute no console python3 caderninho.py
